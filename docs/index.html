System stats,
always on top.

Lightweight overlay for Windows that shows CPU, GPU, RAM, Disk I/O and Network metrics in a transparent bar that stays above any app — including games.

Metrics

CPU % · Freq GHz
Usage percentage and average clock speed across all cores.
GPU % · Temp · Clock
NVIDIA GPU usage, temperature and core frequency via NVML.
GPU Power · VRAM
Power draw in watts and video memory usage in MB.
RAM % · GB
Memory usage as a percentage and absolute value.
Disk I/O MB/s
Real-time read/write throughput via Windows PDH counters.
Network ↓↑ MB/s
Download and upload speed across all network interfaces.

How to use

1

Download and run

Download an8nymous-stats.exe and run it — no installation required. The overlay appears in the top-right corner of your screen.

2

Open Settings from the tray

Right-click the tray icon to open Settings. Choose which metrics to show, reorder them, change colors, font and position.

3

Works above games

The overlay re-asserts HWND_TOPMOST every 500ms, staying visible even when a game window tries to take over. Run your game in Borderless Windowed mode for best results.

4

Start with Windows (optional)

Enable the startup option in Preferences to launch automatically at login.

Requirements

  • Windows 10 or Windows 11 (64-bit)
  • Microsoft WebView2 Runtime — pre-installed on Windows 11, auto-installs on Windows 10
  • NVIDIA GPU for GPU metrics — the app runs normally without one, GPU fields are simply hidden
  • No admin rights required
